# Holyheld — Web3 Debit Card

**Duration:** 2023–2024  
**Role:** Product Designer (UX/UI)  
**Platform:** Web & Mobile (iOS/Android)  
**Website:** [holyheld.com](https://holyheld.com/)

**Key achievements:** Designed 50+ screens with light/dark themes in a minimalist modern style focused on usability. Below are selected screens showcasing the interface.

## Problem & Context
Fintech startup enabling cryptocurrency spending via debit cards. Users faced complex interfaces, security concerns, and limited currency support in existing solutions.

## My Role
End-to-end design process: from research to final mockups and prototypes for web and mobile platforms. Collaborated with product manager and developers.

## Research
Conducted competitive analysis, user interviews, and UX audit of existing solutions. Key insights: need for simplified payment process and enhanced security.

## Design Process
1. Information architecture
2. Wireframes and user flow optimization
3. High-fidelity prototypes
4. Usability testing

## Visual Design
Developed a component system focusing on accessibility and responsiveness. Created unified visual language for web and mobile versions. Designed over 50+ screens featuring both light and dark themes, maintaining a minimalist yet modern aesthetic that prioritizes ease of use and intuitive navigation.

## Results
Created an intuitive interface accessible to both crypto enthusiasts and newcomers. Successfully implemented support for multiple blockchain networks.
